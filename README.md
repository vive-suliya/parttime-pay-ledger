# 직원 급여 관리 앱

식당 직원들의 급여를 관리하는 React Native 앱입니다.

## 주요 기능

1. **직원 관리 (CRUD)**
   - 직원 추가, 수정, 삭제

2. **달력 기반 근무 기록**
   - 달력에서 날짜를 선택하여 근무 기록 추가/수정
   - 슬라이더를 사용한 직관적인 시간 설정

3. **급여 계산**
   - 시급: 13,000원 (모든 직원 동일)
   - 소득세: 3.3% 자동 공제
   - 실시간 급여 요약 표시

4. **월별 급여 내역**
   - "이번달 얼마?" 버튼으로 월별 상세 내역 확인
   - 직원별 근무 시간, 급여, 공제액, 실지급액 표시

## 기술 스택

- React Native (Expo)
- TypeScript
- React Navigation
- AsyncStorage (로컬 데이터 저장)
- react-native-calendars

## 설치 및 실행

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치
```bash
npm install
```

### 실행
```bash
# 개발 서버 시작
npm start

# Android 에뮬레이터에서 실행
npm run android
```

## 배포

### Expo Go로 테스트
1. Expo Go 앱을 스마트폰에 설치
2. `npm start` 실행 후 QR 코드 스캔

### APK 빌드 (핸드폰 설치 파일 만들기)

**가장 쉬운 방법:**
```bash
./build-apk.sh
```

**또는 수동 빌드:**
```bash
# 1. Expo 계정 로그인 (무료 계정 생성 가능)
eas login

# 2. 프로젝트 설정
npm run build:configure

# 3. Android APK 빌드
npm run build:android
```

빌드 완료 후:
- 다운로드 링크가 제공됩니다
- APK 파일을 다운로드하여 핸드폰에 설치하세요
- 자세한 내용은 `BUILD_INSTRUCTIONS.md` 파일을 참조하세요

## 사용 방법

1. **직원 등록**
   - 메인 화면 우측 상단 "직원 관리" 버튼 클릭
   - "+ 추가" 버튼으로 직원 추가

2. **근무 기록 입력**
   - 메인 화면 달력에서 날짜 클릭
   - 직원 선택 및 근무 시간 설정 (슬라이더 사용)
   - 저장 버튼 클릭

3. **급여 확인**
   - 메인 화면 하단에서 총 급여 및 공제액 확인
   - "이번달 얼마?" 버튼으로 상세 내역 확인

## 데이터 저장

모든 데이터는 로컬 스토리지(AsyncStorage)에 저장되며, 앱을 삭제하지 않는 한 데이터가 유지됩니다.
